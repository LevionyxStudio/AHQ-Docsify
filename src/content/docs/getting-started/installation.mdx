# 🚀 Installation

Follow these steps to get **AHQ Docsify** running on your local development environment.

### 🔧 Step 1: Clone the Repository

```javascript
git clone https://github.com/LevionyxStudio/AHQ-Docsify.git
```

### 📁 Step 2: Navigate to the Project Directory

```javascript
cd AHQ-Docsify
```

### 📦 Step 3: Install Dependencies

```javascript
npm install
```

### 🧪 Step 4: Run the Development Server

```javascript
npm run dev
```

### 🌐 Step 5: Open in Your Browser

Open your browser and visit:

```javascript
http://localhost:3000
```

> ✅ That’s it! AHQ Docsify is now running locally. You can start editing `.mdx` files inside the `/docs` folder and see live updates immediately.
