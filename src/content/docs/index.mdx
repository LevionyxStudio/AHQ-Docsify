# Welcome to AHQ Docsify

**AHQ Docsify** is a fast, modern, and fully customizable documentation site generator built with Next.js and Tailwind CSS.

Whether you are building product docs, API references, or internal wikis, AHQ Docsify helps you write, manage, and deploy beautiful documentation effortlessly.

## Key Features

- ⚡ **Fast and performant** with Next.js
- 🎨 **Beautiful responsive design** powered by Tailwind CSS
- 📚 **MDX support** allows mixing Markdown and React components
- 🌍 **Built-in search and navigation** for great user experience
- 🔧 **Easy customization** through theming and plugins

To get started, check out the **Getting Started** section where you'll find setup instructions and examples.
