# 🎉 AHQ Docsify v1.0.0 — First Official Release

We’re proud to announce the **first official release** of **AHQ Docsify** — a fast, modern, and fully customizable documentation site generator built with **Next.js**, **Tailwind CSS**, and **MDX**.

This milestone marks the beginning of a clean, developer-focused approach to building beautiful and performant documentation sites.

---

## 🚀 What's Included

### 🔍 Built-in Search  
Easily searchable docs powered by a lightweight and extensible search system.

### 💡 Component Previews  
Display live previews of your React components directly in the documentation using the `<PreviewCode>` component.

### 🎨 Theme Customization  
Full control over theming with Tailwind. Change colors, fonts, layout, and more — no hassle.

### 📁 Organized Structure  
Your content is neatly organized and easy to scale. Folder-based structure keeps everything clean and maintainable.
